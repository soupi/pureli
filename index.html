Installation

From source:

Install GHC 7.8.* and cabal and run the following commands:

git clone https://github.com/soupi/pureli
cd pureli
cabal sandbox init
cabal install

HOWTO

Read manual/HOWTO.md for information on how to run Pureli programs and REPL.

Examples

Hello World

(module main)

(define main
  (print! "Hello World!"))

Echo

;The program will repeat anything the user writes until ^C
(module main)

(module "stdlib/std.pli" std (bind!))

(define main
  (do!
    [print! "echo program"]
    [letrec ([go!
      (lambda ()
        (do!
          [bind! (read!) print!]
          [go!]))])
      (go!)]))

Exponent

(module main)

(require "stdlib/std.pli" list)
(require "stdlib/std.pli" std)


(define main
  (do!
    (print! "calculating 2^10000 + 2^10001 + ... + 2^10030")
    (let! result (pure calculation))
    (print! result)))


(define calculation
  (list/reduce +
    (list/map
      (std/curry exponent 2)
      (list/map +
                (list/replicate 30 10000)
                (list/range 1 30)))))


(define exponent (n expn)
  (if (<= expn 0)
      1
      (* n (exponent n (- expn 1)))))
