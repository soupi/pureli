    Pureli is

    • Pure

      Pureli enforces the separation of effectful and effectless code and disallows mutable state

    • Functional

      Loops and classes are replaced with recursion, modules and higher order functions

    • Parallel Evaluated

      Automatic parallelization of code deterministically and with no race conditions

    • Dynamically Typed

      The types of values are checked at runtime. Pureli can parallelize code without static analysis

    • Lisp

      Pureli's syntax is inherited from the long tradition of the LiSP family of programming languages

    Installation

    From source:

    Install GHC 7.8.* and cabal and run the following commands:

    git clone https://github.com/soupi/pureli
cd pureli
cabal sandbox init
cabal install

    Examples

    Hello World

    (module main)

(define main
  (print! "Hello World!"))

    Echo

    ;The program will repeat anything the user writes until ^C
(module main)

(module "stdlib/std.pli" std (bind!))

(define main
  (do!
    [print! "echo program"]
    [letrec ([go!
      (lambda ()
        (do!
          [bind! (read!) print!]
          [go!]))])
      (go!)]))

    Exponent

    (module main)

(require "stdlib/std.pli" list)
(require "stdlib/std.pli" std)


(define main
  (do!
    (print! "calculating 2^10000 + 2^10001 + ... + 2^10030")
    (let! result (pure calculation))
    (print! result)))


(define calculation
  (list/reduce +
    (list/map
      (std/curry exponent 2)
      (list/map +
                (list/replicate 30 10000)
                (list/range 1 30)))))


(define exponent (n expn)
  (if (<= expn 0)
      1
      (* n (exponent n (- expn 1)))))

    Unevaluated parameters

    (module main)

(define first-element (~x)
  (eval (car 'x)))
